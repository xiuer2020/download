远程仓库
本地生成ssh key		->	ssh-keygen -t rsa -C"yourmail@example.com"
添加远程仓库		->	git remote add origin git@github.com:xxx
删除远程仓库		->	or:git remote remove <name>

提交到远程仓库master分支	
1.git push -u origin master
2.git push
3.-u指定默认远程仓库
4.--set-upstream 提交其他分支

提交多个分支到远程仓库	->	git push --set-upstream origin dev

从远程仓库拉取数据
1.git pull master
2.--set-upsteam拉取所有分支

克隆远程			->	git clone git@github.com:xxx

分支操作
查看分支
1.git branch
2.查看远程仓库所有分支	->	git branch -r 

创建分支			->	git branch xxx
切换分支			->	git switch xxx
合并分支			->	git merge xxx

删除分支
1.git branch -d xxx
2.删除远程仓库某个分支	->	git branch -r -d xxx

查看分支合并图		->	git log --graph

git常用命令
查看当前仓库状态		->	git status

查看文件区别
1.比较工作区与暂存区	->	git diff
2.git diff readme.txt
3.git diff<文件>HEAD
4.git diff commitld readme.txt
5.git diff commitld commitld

查看版本历史		->	git log --pretty=oneline
查看操作历史		->	git relog

版本回退
1.回退到上个版本		->	git reset HEAD
2.回退到某个版本		->	git reset commitld
3.强制替换当前工作区	->	--hard

撤销更改

丢弃工作区改动
1.git restore file
2.git restore
3.git checkout--<文件>

丢弃暂存区更改
1.git restore--stageed file
2.git restore --staged
